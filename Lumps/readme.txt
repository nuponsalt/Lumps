ディスクからデスクトップ上にデータをコピーして【WindowsGame.exe】をダブルクリックしてください。

このゲームは、3Dアクションゲームです。(注、音があります。)
上から降ってくるカラフルな球を、制限時間いっぱい自機にくっつけて
大きな塊をつくるのがこのゲームの目的です。

// ------------------------------------------------------------------
// 操作方法
// ------------------------------------------------------------------

//* タイトル画面 *//
左右（←→）キー 	: 選択
Enter/Z		: 決定
BackSpace/X	: 戻る

//* プレイ画面 *//
A		: カメラ左回転
D		: カメラ右回転
方向キー		: 左右前後移動

//* リザルト画面 *//
Enter		: タイトルに戻る
